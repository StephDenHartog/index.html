My Web Design Work

Week 4 – Navigating Pages

This was my first multi-page site. I learned how to connect pages using relative links and create navigation menus.

Week 5 – Using an External Style Sheet

During this week, I moved my styling into one shared CSS file. I practiced keeping things consistent across pages — colors, fonts, and layout spacing — which made updates so much easier.

Week 6 – Light and Dark Mode

I added light and dark themes using CSS variables and media queries. It was cool seeing my site automatically switch colors depending on system settings.

Week 7 – Interactive Settings Panel

This project taught me how to make a web page feel interactive. I built a “notification settings” panel that changed on hover and active states. It gave me a better idea of how user interfaces work.

Week 8 – Box Model Product Cards

For this assignment, I created a few modern product cards using box-sizing: border-box. I adjusted margins and padding for spacing and used a soft shadow instead of borders to give the cards a clean look.
